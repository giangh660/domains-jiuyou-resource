# domains-jiuyou-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，可作为独立网页直接访问。仓库本身不针对任何特定域名或具体网站，仅作为通用页面存档与展示用途。

## 目录结构

```
domains-jiuyou-resource/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
└── README.md       # 本说明文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名即页面标识。
- `assets/`：存储页面所需的样式、脚本、字体、图像等静态资源，按需组织子目录。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 页面之间互不关联，可单独访问。
- 归档内容可能包含信息展示、工具页面、测试页面等，具体用途见各页面内部说明。
- 页面内容可能随时间更新或替换，历史版本可通过 Git 提交记录查看。
